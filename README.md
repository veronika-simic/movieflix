# Movieflix

A simple React + TypeScript app that connects to a movie API and lets you **favorite or unfavorite movies**.

## Features

- Browse movies from an API (e.g., TMDB).
- Add or remove movies from your favorites.
- Favorites are saved in `localStorage` so they persist across sessions.
- Simple, responsive UI with movie cards.

## Technologies

- React
- TypeScript
- Context API for state management
- CSS for styling
